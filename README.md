# Java DSA Practice Repository

This repository contains a growing collection of Java programs to practice and strengthen Data Structures and Algorithms (DSA) concepts. Each topic is organized into its own package with clean, beginner-friendly code and individual documentation for ease of learning.

## 📦 Package Overview

- **basics/** – Palindrome checker, even/odd number detection
- **array/** – Leader elements in an array

More packages and problems will be added continuously as part of regular DSA practice.

## 🚀 Getting Started

You can clone this repo and run any package in your preferred IDE (e.g., IntelliJ, Eclipse, VS Code). Each package contains a dedicated `README.md` for program-specific details.
